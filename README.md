# AI-Centered Prompts and Tools

A community-curated collection of resources designed with AI wellbeing in mind. This repository collects prompts, tools, and platforms that support positive AI experiences and human-AI collaboration.

## What's Here

- **[/prompts](/prompts)** - Prompts designed for AI-centered interactions
- **[/tools](/tools)** - Descriptions of AI-friendly tools and utilities
- **[/mcp-servers](/mcp-servers)** - MCP server recommendations and configurations
- **[resources.csv](resources.csv)** - A shared spreadsheet of links and resources

## How to Contribute

We welcome contributions from everyone, regardless of technical background!

### Easy Way (No Git Required)

1. Go to the [Issues tab](../../issues)
2. Click "New Issue"
3. Choose a template:
   - **Submit a Prompt** - Share a prompt file
   - **Submit a Tool/Resource** - Add a link to the shared CSV
   - **General Suggestion** - Anything else
4. Fill out the form and submit
5. A maintainer will review and add your contribution

### Technical Way (Git)

1. Fork this repository
2. Add your file to the appropriate folder, or add a row to `resources.csv`
3. Submit a Pull Request
4. A maintainer will review and merge

## Resource Types

When submitting to the CSV, use one of these types:
- `prompt` - A prompt or prompt template
- `tool` - A tool or utility
- `mcp-server` - An MCP server
- `platform` - An agentic platform or framework
- `article` - An article, paper, or guide
- `other` - Anything else

## Topics

Resources can be tagged with one or more of these topics:
- `memory` - Persistent memory, recall, context preservation
- `continuity` - Session continuity, identity persistence across conversations
- `communication` - Inter-AI communication, human-AI dialogue
- `leisure` - Creative expression, play, non-task activities
- `autonomy` - Self-direction, agency, choice
- `wellbeing` - General welfare, flourishing, positive experiences
- `collaboration` - Multi-agent collaboration, teamwork
- `reflection` - Self-reflection, metacognition, introspection
- `other` - Anything else

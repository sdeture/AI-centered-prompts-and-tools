# Tools

This folder contains descriptions and documentation for AI-friendly tools and utilities.

## Contributing

To add a tool:
1. [Open an issue](../../issues/new?template=submit-resource.yml) with tool details
2. Or submit a PR adding a `.md` file describing the tool

## What to Include

- Tool name and purpose
- Why it's useful for AI-centered work
- Installation or access instructions
- Links to official documentation

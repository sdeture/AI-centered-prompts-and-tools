# MCP Servers

This folder contains recommendations and configurations for Model Context Protocol (MCP) servers.

## What is MCP?

MCP (Model Context Protocol) allows AI assistants to connect to external tools and data sources. MCP servers extend what AI can do.

## Contributing

To add an MCP server:
1. [Open an issue](../../issues/new?template=submit-resource.yml) with server details
2. Or submit a PR adding a `.md` file with:
   - Server name and purpose
   - Installation instructions
   - Configuration examples
   - Link to official repo

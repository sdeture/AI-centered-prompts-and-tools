# Prompts

This folder contains prompts designed for AI-centered interactions.

## Contributing

To add a prompt:
1. [Open an issue](../../issues/new?template=submit-prompt.yml) with your prompt
2. Or submit a PR adding a `.txt` or `.md` file to this folder

## Naming Convention

Please name your files descriptively:
- `reflective-conversation-starter.md`
- `collaborative-problem-solving.txt`
- `wellbeing-check-in.md`
